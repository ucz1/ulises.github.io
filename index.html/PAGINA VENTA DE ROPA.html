Sobre el proyecto

Este modelo de tienda fue desarrollado para mostrar un sistema de venta sin stock (Print-on-Demand). La producción y envío se realizan bajo pedido, reduciendo costos y desperdicio textil. Ideal para marcas emergentes o presentaciones empresariales.

  • Hosting gratuito (GitHub Pages)
